# AI Marketing Intelligence System

An end-to-end AI-powered marketing analytics system built on real e-commerce 
transaction data. The system segments customers using K-Means clustering, 
predicts churn using Random Forest, builds a product recommendation engine, 
performs sentiment analysis, and presents all insights through an interactive 
Power BI dashboard.

## Dataset
- **Online Retail Dataset** — UCI Machine Learning Repository  
- Link: https://www.kaggle.com/datasets/vijayuv/onlineretail
- Period: December 2010 – December 2011
- Records: 541,909 transactions, 4,372 customers
